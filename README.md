# Restaurant Ordering System - Demo Front-End with SvelteKit

## Installation

- ```console
  git clone git@github.com:nniculae/ros-front.git
  ```
- ```console
  cd ros-front
  ```
- ```console
  npm install
  ```
- ```console
  npm run start-db
  ```
- ```console
  npm run dev -- --open
  ```
- Go to:
  ```console
  http://localhost:5173/
  ```
